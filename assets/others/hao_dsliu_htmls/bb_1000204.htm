【转】fbKindle：让Kindle变身ePub阅读器

发布时间：2011-7-21 10:35
分类名称：Kindle


ePub作为电子书的标准格式并大规模应用是Barnes & Noble书店，国际知名的sony、nook和汉王是原生支持ePub的。Google也制作了大量的ePub图书，

从而将ePub的版权应用范围放大不少；当然Google的战略目标是amazon，因为Kindle原生不支持ePub这种广为流传的格式。（很古怪的是，很多强大的图书管理软件包括calibre也不支持ePub。）

对于读书人来说，面对庞大的ePub资源岂可浪费？

俄罗斯人脑容量的确了得，移植FBReader（FBReader — e-book reader for Unix/Windows computers，http://www.fbreader.org/）制作了fbKindle，它使在Kindle上阅读ePub格式电子书成为可能。

另外，fbKindle还支持mobi和prc格式。

在kindle键盘上使用fbKindle的主要快捷键:

L = 显示书架
O = 软件首选项设置
F = 查找
P = 查找上一个
N = 查找下一个
Home = 开始处/封面
A = （向书架）添加图书
I = 图书信息
R = 旋转屏幕
Alt/Back = 退出fbKindle
Alt/Shift = 在Kindle Home和fbKindle之间切换（也就是说使用kindle原生系统和fbKindle是不需要重启机子的，快速切换）
G = 跳转到某页
Menu = 唤出fbKindle菜单
Space = 刷新屏幕
Shift按三次 = 屏拷图片（图片自动保存在用户根目录下screenshot.bmp，只可保存一张图片，后续屏拷操作自动覆盖这个文件，不像在Kindle自身系统下是自增的。）
音量-+ = 缩小或放大文字

一、fbKindle的安装
1、安装快捷键管理软件Launchpad；
2、安装fbKindle。
更详细的介绍不用重复了，请移步浏览《江郎才无尽！kindle 3 高级折腾教程！！》http://bbs.mydoo.cn/thread-19025-1-1.html

二、fbKindle的启动
在Kindle Home中按Shift/f/r启动fbKindle。

三、fbKindle的设置
主要说一下让fbKindle支持中文：Menu 键唤出fbKindle菜单（或者直接按O键），找到Language标签，设置Default Language为Chinese，Default Encoding Set为East Asia，Default Encoding为Chinese Simplified GBK。Alt/Back退出fbKindle。
拷贝一个中文字体（通常用微软雅黑或者文泉驿微米黑，字体不发虚，当然也可以是你喜欢的其他中文字体）到kindle用户目录fbKindle\qtKindle\lib\fonts下，重启kindle。
Shift/f/r，fbKindle将支持中文。





四、fbKindle的使用流程
在fbKindle按A，跳出系统目录/mnt/us/...，查找图书（在/mnt/us/documents下），选中并打开图书时会跳出图书信息设 置标签，通常不用管它（也许你需要设置图书文字编码，待阅读时出乱码再回头调整吧）。按L，显示书架，按上下方向键至图书标题下的read book，按确认，开始阅读图书。
