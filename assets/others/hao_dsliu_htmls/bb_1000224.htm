名词就不解释了，大家跟着我，一步一步来，绝对没问题的！首先说明本教程XP，Win7都适用。

一：准备工作

1；电脑下载MOTO驱动并安装。Handset_USB_Driver_32_v4.8.0.rar (大小1631k)

2；电脑下载刷机工具并安装。rsdlite_4.9.rar(大小4347k)

3；下载最新刷机包备用。这里提供DK手机论坛4月9日首发的ME722-MLS2GC_U6_2.6.0刷机包  （注意：刷机包不要更改中文名，放在你的电脑你能找到的任意盘内，盘符不能有中文名） 

优蛋下载地址：http://u.115.com/file/f466acd8e0


提示：刷机前的一些要做的工作：备份好你的电话簿，应用程序，拔掉sim卡，拔掉内存卡（也可以不做这个工作）确保以上准备工作已经就绪，手机电量充足，电脑不会断电关机，手机上一些需要的资料已经备份，就可以开始更新你的爱机系统了。

1；清除手机用户数据（此步也可以不做，建议还是清除一下）方法：手机关机的状态下按住手机键盘的 “X”键不放，在按开机键8秒左右，直到手机进入一个叹号的标志和一个小机器人的界面的时候按键盘的“@”键 进入所谓的recovery模式，用键盘上下键选择“wipe data/factory reset”按OK键，选择YES -- delete all user data 点OK，看到…………要不停地点屏幕上的ok只到完成；用键盘上下键选择“wipe cache partition” 点OK，看到…………要不停地点屏幕上的ok直至完成。

说明： wipe date/factory reset意思是：清除手机系统存储区中的所有数据，进行恢复出厂设置
            wipe cache partition意思是：清除手机缓存分区中的所有数据。
            reboot system now意思是：立即重启手机

2；.进入Milestone2的BootLoader模式：
  手机在关机状态下，按住“音量+键”和“拍照键”不放，再按下“电源键”，保证此时三个键都被按下，直至屏幕亮起松手，然后会显示黑底白字的如下状态：

Bootloader
70.12

Battery OK
OK to Program
Connect USB
Data 
Cable

3；将手机插上USB数据线连到电脑上，打开RSD4.9，电脑会识别出里程碑2手机的各种设备并自动安装驱动，最后RSD Lite软件会显示出1号USB端口的设备Connected，如下图：
 
手机上的信息也变为
Bootloader70.12

Battery OK
OK to Program
Transfer Mode：
USB

4；在RSD软件中点“选择”按钮，找到刷机包，点打开，然后点“开始”按钮就开始刷机了，
 

等一会，progress下面的百分比就开始走了，

手机信息也相应的变为


SW UPDATE 
INPROGRESS


也就是软件更新进行中，等到100%时，手机会自动重启，不用管，等第二次达到 100%时，如果是欧版刷行货ROM，RSD会显示“please manually power up this phone”也就是提示你手动重启手机，而此时手机启动卡在Blur开机动画处，此时拿出电池，过5秒再安上

5；再次清除数据（这一步和 第一步是一样的步骤）。推开全键盘，按下“X”键不放，按住开机电源键，保持两个键被同时按下的状态，直到出现M标志后，松手，此时手机会停留在一个叹号 机器人的画面下，这时按下键盘最下面一排左起第四个的“@”键，即可进入恢复程序选单界面，用全键盘的导航键，把光标定在“wipe date/factory reset
”这一行上，按键盘上的“OK”键，选择“YES”那行，再按“OK”键，等会返回到选单界面，同理，光标 定到“wipe cache partition”这一行上，按“OK”键,再等返回到选单界面，在“reboot system now”这一行上，按“OK”键，手机重启，等待进入桌面，刷机完成！

说明： wipe date/factory reset意思是：清除手机系统存储区中的所有数据，进行恢复出厂设置
            wipe cache partition意思是：清除手机缓存分区中的所有数据。
            reboot system now意思是：立即重启手机
