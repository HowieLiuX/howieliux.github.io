转载：如何讓ThinkPad的TrackPoint的中鍵支援MSN、FireFox、IE7和Office 2007的捲動? (NB) (ThinkPad)

发布时间：2010-1-19 12:15
分类名称：Others


 Abstract
ThinkPad的TrackPoint相當好用，可惜預設並不支援MSN、Office 2007和IE7，而FireFox雖可垂直捲動，卻無法水平捲動，該如何讓TrackPoint支援MSN、FireFox和Office 2007呢?

Solution
Step 1：
若為ThinkPad，用Notepad開啟c:\WINDOWS\system32\tp4table.dat。
若為ThinkPad USB Travel keyboard with UltraNav，用Notepad開啟c:\Program Files\Synaptics\SynTP\tp4table.dat。

Step 2：
在
;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;
; Pass 0 rules (These rules run first)
;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;

區段中的最後，加入以下文字
; MSN Messenger 
*,*,msnmsgr.exe,*,*,*,WheelStd,0,9 

; IE 7 
*,*,iexplore.exe,*,*,*,WheelStd,0,9 

; Firefox 
*,*,firefox.exe,*,*,MozillaWindowClass,WheelVkey,0,9 

; Word 2007 
*,*,winword.exe,*,*,*,WheelStd,0,9 

; Powerpoint 2007 
*,*,POWERPNT.EXE,*,*,*,WheelStd,0,9 

; Outlook 2007 
*,*,outlook.exe,*,*,*,WheelStd,0,9 

Remark
1.將來若有新軟體不支援TrackPoint，都可以自己利用這種方式修改。
2.ThinkPad USB Travel keyboard with UltraNav在開機時需要用到三個檔案，分別是
C:\Program Files\Synaptics\SynTP\SynTPLpr.exe
C:\Program Files\Synaptics\SynTP\SynTPEnh.exe
C:\Program Files\Lenovo\UltraNav Keyboard\SkdUNav.exe
這三個檔案不能用Startup Delayer去管理，否則tp4table.dat所加的程式，如MSN則無法支援。
