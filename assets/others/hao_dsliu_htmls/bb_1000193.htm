Kindle 3原版系统中，自带光标移动屏幕取词翻译。配上英汉字典，遇见生词随时查询中文含义非常实用。我想，每个人可能都有自己需要的词典，不论英英、英汉、英法、还是汉语词典，但很多刚入手Kindle的朋友对于词典的使用会遇到很多问题。

kindle3中怎么用词典

在原生系统下，看英语小说，阅读英文文档过程中遇到不认识的英文单词，如何直接查词典而不用退出正在阅读的界面呢？Kindle 3中启动词典，实际就像在电脑中用了翻译工具一样，光标指到那个词上，即时翻译。

  • 方法：直接按方向键，把光标移动到需要查询的单词前，在屏幕的最上方或最下方就出现释义了，其中"More和回车键"它是告诉你如果想看更详细的解答，请按回车键。
  • 注意：上述操作如并没有调出词典，而只出现加标签和注释提示，可能是因为此文件格式不支持词典。支持的词典，如阅读.mobi时，按方向键就自动激活词典了。
  • 解决办法：把文件转成mobi的格式，或者发给Amazon，它会转换成azw格式的文件，就可以用词典了。

Kindle3 怎么安装英汉词典

Kindle3内置词典是两个英英词典。把Kindle3当做学英语的工具在深入学习单词用法时比较有用，对于阅读小说看新闻，查某词的中文解释，需要自己上传mobi格式英汉词典到Kindle3中。

  • 方法：下载到.mobi英汉词典后，同装书一样，接上USB，把.mobi文件复制到Kindle 3的documents目录下，即可和Kindle 自带英英词典一样使用划词翻译功能了。只不过需要把它设置成默认的词典。
  • 注意：原装系统只支持mobi格式的词典，mobi格式的词典也只能在原生系统下使用，多看系统下不支持。
  • .mobi词典，本身就相当于一本本的电子书。在原系统书目下，可以以打开电子书的形式直接打开词典，并在其中浏览并输入单词查询。
  • 多看系统添加词典的方法，Scomper打开了方便之门。下载压缩包—解压—找到对应的词典文件夹—复制完整的三个文件idx 、ifo、 dict，拷贝到DK_System\system\dict下即可。

怎么设置一个词典作为默认词典

按Home键进K3原生系统 -> Menu -> Settings -> Menu -> Change Primary Dictionary，就会出现词典的列表，前提是你已经下载并上传一些常用的词典到Kindle 上了，选择需要的词典，确定即可设为默认词典。

哪里有词典免费下载呢

有的淘宝店销售Kindle 3时就会附赠了一些资源，比如漫画、词典等。网络上也有大把的资源。

  • 多看系统下可用资源：英汉词典，个人觉得装上2个足够用了朗道英汉字典+牛津现代英汉双解词典。另外，Stardict 81部中文词典下载中有非常丰富的词典资源。
  • 原系统下可用资源：20本mobi格式的词典。
  • 注意：在原系统下的词典，在不更改系统设置的情况下，是不会在词典中显示中文的。需要用到changelocale命令，来解决Kindle 3原生系统的中文支持问题1。

有人问这些词典是否只在Kindle3中能用，其实只要是支持mobi格式的电子书，尤其是Kindle2,KindleDX也都支持这些词典的。

在线"词典"快速查词的方法

我们在非阅读模式下，需要用Kindle 3设备来查词，联网情况下可以使用@命令启动某些快捷键。

  • 方法：点击Del键，底部弹出搜索框。点击Sym，然后选择输入@命令，几种方法：

@dict+词语（从词典中快速查词）
@web+词语（在Google中搜索该词）
@wiki+词语（在维基百科中搜索该词）

  • 注意：上述命令中，+号等于空格。输入完某词后，再次按Sym键，则去掉符合框，即可查询了。
