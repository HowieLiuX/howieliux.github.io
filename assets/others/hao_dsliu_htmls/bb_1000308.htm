Win7 不为人知的一些快捷键

发布时间：2010-3-21 13:07
分类名称：windows


  1. Ctrl+Alt+Tab 鼠标选择
  2. 按左 Alt+左 Shift+PrtScn（或 PrtScn） 启用或关闭高对比度
  3. Shift+F10 显示选定项目的快捷菜单
  4. 徽标键 + Pause 系统属性
  5. 标键 + T 循环切换任务栏上的程序。
  6. 徽标键 + 数字 启动锁定到任务栏中的由该数字所表示位置处的程序。如果该程序已在运行，则切换到该程序。
  7. Shift + 徽标键  + 数字 启动锁定到任务栏中的由该数字所表示位置处的程序的新实例。
  8. Alt + 徽标键  + 数字  打开锁定到任务栏中的由该数字所表示位置处的程序的跳转列表。
  9. （ctrl）徽标键  + Tab Aero Flip 3-D
  10. 徽标键  + B 切换到通知区域
  11. 徽标键  + 空格 预览桌面
  12. 徽标键  + 向上键 最大化
  13. 徽标键  + 向下键 最小化
  14. 徽标键  + 向左键 最大化到左侧
  15. 徽标键  + 向右键 最大化到右侧
  16. 徽标键  + home 最小化除活动窗口外的所有窗口
  17. 徽标键  + ‘+’/ '-' 放大，缩小
  18. 徽标键  + Esc 退出放大
此处摘自win7帮助中心，更加详细的可以参考win7帮助中心，里面有更加全的快捷键。
