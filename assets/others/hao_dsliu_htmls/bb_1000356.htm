前几天装了个mac,打算玩玩iphone程序开发, 刚刚装上适应俺半天,最起码都ctrl - c,都不能用了,郁闷都我.呵呵,还好,过了一4天,基本上都熟悉了, 而且以前经常完Linux各种版本, 对这种系统摸得也比较熟了.

题 目看上去好像比较多此一举, 因为大家都知道mac下realplayer已经可以播放了.的确是可以,但是它一是不稳定,而是渲染出来都图像怪怪的,(其实我就开了一个*片, 一开就崩了.嘿嘿)而且在我那4年前配置都机器上面跑的很卡,屏幕放大后有马赛克一样都格子.(我就觉得纳闷, rmvb是它官方都东西,怎么做的这么不中用,反而用mplayer很流畅, 播放效果也好)

首先必备都工具:

1. Realplayer 11, 可以到官方下载.记得最好用迅雷下载,不然会把你逼疯了,那网速是超级都慢.毕竟人家国外不考虑我们都国情.呵呵.最好还是到国内下载, 推荐一个,去太平洋: http://download.pchome.net/multimedia/media/detail-63260-0.html

2. Mplayer, version: MPlayer OS X 1.0rc2(注意版本是1.0rc2, 不是2.0), 官方地址: http://www.mplayerhq.hu/MPlayer/releases/MPlayerOSX_1.0rc2.dmg, 或者自己搜索.

好了, 之前看其他blog有说过还用其它工具, 这里就不用了, 多敲入几个命令就解决了:

首 先把Realplayer, Mplayer 都drag到applications里, 然后右击Realplayer, 选择"Show package contents", 然后就打开这个realplayer.app这个文件夹了, 然后定位到这个目录RealPlayer.app/Contents/Frameworks/HXClientKit.framework /HelixPlugins/Codecs, 将Codecs文件夹托出来, 就放到根目录(/)底下吧, 然后将其名称重命名为小写的: codecs

然后, 打开/Applications/Utilities/Terminal, 进入命令模式操作:

首先得进入root用户, 如果你第一次进入root, 那么先设置个密码:

sudo passwd root

好像接下来让你先输入当前系统都密码, 然后再输入两次新密码.

接着输入命令:

su -

系统要求输入root密码, 记住是你刚刚设置都那个密码.输入密码后,如何成功了,就会提示符都前缀显示root名称了,如果不正常,那就是提示: su: Sorry

进入root后, 下面都是输入命令:

cd /usr

ls

找找有没有下面有没有local目录, 如果没有:

mkdir local

接着

cd local

mkdir lib

cd lib

cp -R /codecs . (注意, 这个目录名都是小写的, 如果不是, 先把他重命名,比如之前是大写的,那就: mv Codecs codecs

cp这句是说将刚刚托出来都 Codecs复制到/usr/local/lib目录底下(如果你下载的mplayer 是2.0都,那么重命名目录: mv Codecs mplayer

好 了, Over! 试试开一个rmvb都文件看看, 是不是不卡了,放大了也不会有格子, 有个建议,就是实时播放都时候拖动好像会有一段延迟.导致图像和音频不同步, 不过我试了一下,只是暂时都几秒钟, 或者你可以先暂停了,再调整窗口大小.(再开开那个叉叉片, 就好了.估计是字符编码引起都问题吧.)
